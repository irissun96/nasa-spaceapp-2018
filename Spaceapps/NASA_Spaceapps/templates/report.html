Vigil Fire Prevention System

This system is designed to be a web-based app which allows the user to upload fire occurences near and exam the possibility of fire in every area in the world. The embedded algorithm uses the data from NASA and users to analyse and predict the fire hazard.

Data Sources
  • NASA OpenNEX Appstore
  • CLIMATEDATA.US
  • Fire Information for Resource Management System
  • ADS-B Exchange
